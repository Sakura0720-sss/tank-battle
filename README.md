# 🎮 坦克大战 (Tank Battle)

一个经典的 HTML5 Canvas 坦克大战游戏，灵感来自经典的 NES 游戏《Battle City》。

## 🕹️ 玩法

- **WASD / 方向键** — 移动坦克
- **空格 / J 键** — 发射子弹
- **手机端** — 触屏滑动移动，自动射击

## 🎯 游戏规则

- 消灭所有敌方坦克即可过关
- 保护你的基地（🦅），基地被摧毁则游戏失败
- 生命值耗尽则游戏结束
- 击毁砖墙可以开辟新路线
- 钢铁墙壁无法被摧毁
- 敌方坦克有几率掉落道具

## ✨ 特性

- 🎨 精美的像素风格画面
- 🧱 可破坏的砖墙和不可破坏的钢铁墙
- 💥 爆炸粒子特效
- 🎁 道具系统（加速射击、护盾、冰冻、炸弹）
- 📈 3个不同关卡，难度递增
- 📱 支持手机触屏操作
- 🏆 分数系统

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可游玩！

```bash
# 或者使用任意 HTTP 服务器
npx serve .
```

## 🔧 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 无需任何依赖
